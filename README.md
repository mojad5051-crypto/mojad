# Florida State Roleplay Staff Training Website

This folder contains the official static website for Florida State Roleplay staff training.

## Files
- `index.html` — the full website page with rules, training guide, driving test, and mod call procedures.

## How to use
1. Open `index.html` in a browser to view the website locally.
2. To publish it as a public website, host the `florida-rp-site` folder on any static site hosting service.

## Recommended hosting options
- GitHub Pages
- Netlify
- Vercel
- Any web server that serves static HTML files

## Notes
- The site is already styled and responsive for desktop and mobile.
- If you want an official domain, point the domain to your hosting provider and upload these files.
